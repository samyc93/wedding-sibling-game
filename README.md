# Wedding sibling game

A mobile-first wedding puzzle game hosted free with GitHub Pages.

- Hannah: `https://samyc93.github.io/wedding-sibling-game/?person=hannah`
- Joe: `https://samyc93.github.io/wedding-sibling-game/?person=joe`

## Current Hannah flow

1. Intro screen
2. Find the odd animal
3. Phone tilt maze with arrow fallback
4. Repeat the symbol pattern
5. First memory/photo checkpoint
6. Put the wedding saying in order
7. Second memory/photo checkpoint
8. Frost wipe final reveal

The player cannot lose. Wrong answers simply provide another try or reveal the answer and continue.

## Files

- `index.html` – page shell
- `styles.css` – mobile-first design and animations
- `app.js` – game engine, levels and personalised wording
